DEV INSTRUCTIONS TO RUN BTO PROGRAM

1. Execute the main.jar located in src/out/artifacts directory
2. To log in, UserIDs of existing personnel can be found under the src/localdata directory. 
   They are labeled with their respective user types.
3. All passwords by default are set to "password"
4. Please avoid direct manipulation of csv files to avoid unexpected/unhandled errors. 
   Safe manipulation of data is executed through the various functionalites accessible in main.jar

important notes:
if unable to locate file path errors are sent, please enter "../" at the start of the every filepath string
under Init() located in MainApp.java. It may fix those errors.
